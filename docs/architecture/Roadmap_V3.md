## 核心技术点

| 组件 | 技术选型 | 理由 |
|------|----------|------|
| String | embstr+raw+int | 内存优化 |
| Hash | ziplist+hashtable | 平衡内存和性能 |
| List | quicklist | 平衡内存和操作 |
| Set | intset+hashtable | 整数优化 |
| ZSet | skiplist+dict | O(logN)操作 |
| RDB | fork+COW | 非阻塞备份 |
| AOF | 混合刷盘 | 灵活性和性能 |
| 过期 | 惰性+定期 | 平衡CPU和内存 |
| 淘汰 | 近似LRU/LFU | 以精度换性能 |
