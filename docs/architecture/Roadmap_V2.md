## 核心技术点

| 组件 | 技术选型 | 理由 |
|------|----------|------|
| 主从Reactor | epoll LT | Linux原生，够用 |
| 连接分配 | RoundRobin | 简单均衡 |
| 任务队列 | MPSC无锁队列 | 高效无锁 |
| 线程池 | 固定数量 | 避免频繁创建 |
| 内存池 | ThreadCache+Central+Page | 减少碎片 |
| 哈希表 | 分段锁 | 减少锁竞争 |
| 过期 | 惰性+定期 | 平衡CPU和内存 |
