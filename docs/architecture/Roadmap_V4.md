## 核心技术点

| 组件 | 技术选型 | 理由 |
|------|----------|------|
| 集群拓扑 | 去中心化 | 无单点故障 |
| 槽分片 | CRC16%16384 | 均匀分布 |
| 节点发现 | Gossip MEET | 简单高效 |
| 故障检测 | PFAIL→FAIL | 多数派确认 |
| 选举 | priority+offset | 公平竞争 |
| 复制 | 异步+增量 | 高性能 |
| 客户端 | Smart Client | 最低延迟 |
| 重定向 | MOVED/ASK | 渐进迁移 |
