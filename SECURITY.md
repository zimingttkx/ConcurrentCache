## 报告漏洞

如果你发现安全漏洞，请通过以下方式联系我们：

- **邮箱**：2147514473@qq.com

请**不要**在 GitHub Issues 或 Discussions 中公开报告漏洞。

## 响应时间

我们会在 **48 小时**内回复漏洞报告。

## 披露政策

- 我们承诺不对外公开报告者的信息
- ，我们会协调发布修复补丁
- 我们会根据情况决定是否公开披露漏洞

## 致谢

对干报告漏洞的研究者，我们会在得到允许后将其加入致谢名单。
